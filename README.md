# ANJANA-P-S - Full Stack Developer Screening Test

## Coding Language Used: Dart

## Problem-1: Calculator Class
* Implementation of Addition, Subtraction, Multiplication, and Division using a `Calculator` class.

## Problem-2: Odd Series (Fixed Terms)
* Generates the first 'a' odd numbers.

## Problem-3: Odd Series (Conditional Terms)
* Generates 'a' odd numbers if 'a' is odd, and 'a-1' odd numbers if 'a' is even.

## Problem-4: Count Multiples
* Counts numbers in an input list that are multiples of 1 through 9.
